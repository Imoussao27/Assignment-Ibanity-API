# Ibanity Take-Home Assignment

## Goal
Set up a connection to the Ibanity API (`api.ibanity.com`) using the Elixir programming language.

## Assignment

1. Sign up at [Ibanity](https://developer.ibanity.com)
2. Create a team
3. Enable a sandbox application
4. Enable the Ponto Connect product
5. Generate your mTLS certificates
6. Configure your client to use these certificates
7. Request an access token
8. Fetch 10 transactions and display them [API Reference](https://documentation.ibanity.com/ponto-connect/2/api/curl#transaction)

> The use of AI tools is allowed and even encouraged. However, please be prepared to explain the reasoning behind your approach and the decisions you made along the way.
